# Yard Radar

A mobile app connecting yard sale hosts with shoppers nearby. Built with Expo (React Native) + Supabase + Mapbox.

## Overview

Yard Radar helps you discover yard sales in your area and allows hosts to promote their sales to nearby shoppers.

## Tech Stack

- **Frontend**: Expo (React Native)
- **Backend**: Supabase
- **Maps**: Mapbox
